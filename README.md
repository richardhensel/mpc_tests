# MPC Tests

Some experiments with Model Predictive Control of a ground vehicle. 

Built with ROS2 Dashing. 

Visualisation with rviz2.

## Build
Beginning from this directory:
~~~
cd ../
~~~
~~~
colcon build 
~~~

## Run
Beginning from this directory:
~~~
cd ../
~~~
~~~
ros2 launch mpc_tests_launch/launch/mpc_tests_launch.py
~~~
